# Neuronforge

Local-first experiment workspace for model evaluation, prompt testing, proofreading workflows, and controlled step-by-step buildout.

## Core folders

- docs/      project docs and workflow notes
- inputs/    source texts and test inputs
- outputs/   model outputs and comparison results
- evals/     evaluation notes and score sheets
- registry/  model, run, and prompt registries
- prompts/   reusable prompt files
- notes/     working log and observations
