# Review Checklist

## Before judging output
- confirm the correct prompt file was used
- confirm the correct input file was used
- confirm the output file saved completely

## Check protected terms
- all character names preserved
- all place names preserved
- all faction names preserved
- all title or honorific terms preserved
- all special canon terms preserved

## Check proofreading behavior
- grammar improved
- spelling improved
- punctuation improved
- obvious usage errors corrected
- no unnecessary rewrites
- no added commentary
- no visible reasoning
- output returned in the requested format

## Check literary integrity
- tone preserved
- voice preserved
- rhythm mostly preserved
- meaning preserved
- imagery not degraded

## Final judgment
- accept
- accept with notes
- reject
